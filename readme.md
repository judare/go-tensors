# Go Tensors

## Build project:
```bash
go build Tensor.go HadamardProduct.go IndexSelect.go Reshape.go Main.go 
```

# Usage

## Info
```bash
./Tensor --info
```

## Reshape:
```bash
./Tensor r
```

## HadamardProduct:
```bash
./Tensor hp
```

## IndexSelect:
```bash
./Tensor is
```
